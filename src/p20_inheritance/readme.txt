p20_inheritance
父类和子类的特点
1. 父类更通用，子类更具体
2. 继承的特性：（1）子类拥有父类的非private方法、属性（2）子类可以重写父类的方法（3）子类可以定义自己的方法和属性

两种继承方式
3. 继承自类 -> 单继承类型：一个父类可以派生多个子类，但是一个子类不能来自于多个父类，换言之，父类：子类 = 1：N。
4. 此时的继承使用-extends： class 子类 extends 父类 {}

5. 继承自接口 -> 变相的多继承类型
6. 此时的继承使用 -> implements： public class C implements A, B {}

关键字：
7. super与self：super引用到父类，this引用到自身，换言之使用super可以调用父类的成员，使用this可以调用自身的成员
8. final：
        （1）修饰类表示该类不允许被继承 final class 类名{}
        （2）修饰方法名表示改方法不允许被子类重写 public/private... final 返回值类型 方法名(){}
        （3）修饰变量表示不允许被修改

构造函数
9. 子类不继承父类的构造函数，而是调用。
        （1）父类为有参构造，必须在子类中 显式的通过super关键字 调用匪类的构造器并配备参数列表【from菜鸟】 ？实际证明不写super也没报错
        （2）父类为无参构造，不需要使用super关键字，系统会自动调用父类的无餐构造器
        （3）如果父类只有有参构造函数，那么子类也不可以定义无参构造函数
        （4）父类可以不定义任何构造函数，这不影响子类定义任何构造函数
        （5）这一块不用太纠结，就按照标准方法写就行。基于 实例化对象 反推 构造函数，然后反推对父类的构造诉求，例如需要无参构造，那么至少父类不能只有有参构造
