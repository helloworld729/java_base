1、局部变量
    位置：方法或者语句块中
    周期：方法执行时被创建，执行完成后销毁
    是否可以被访问修饰符修饰：否
    是否具有默认值：否

2、实例变量/成员变量
    位置：类中 方法外
    周期：与对象的创建和销毁同步
    是否可以访问修饰：是
    是否具有默认值：是

3、类变量/静态变量
    位置：类中 方法外
    周期：第一次访问时被创建，程序结束时销毁
    是否可以访问修饰：是
    是否具有默认值：是
    other：无论实例化多少次，类变量只会维护一份拷贝

4、final
    修饰类：final修饰的类不能被继承
    修饰方法：方法锁定，以防止任何继承类修改它的定义
    修饰变量：final修饰基本变量，该变量初始化后不能再修改；final修饰引用变量后，该变量不能再指向别的对象
